How it's possible

How we train frontier models at a fraction of the cost.

We own every stage — data, architecture, training — so our only cost is compute, and quality compounds.

DATA · Own the data at source

  • 3 owned streams: ~3–4M hrs internet · ~150K hrs human-collected · live app speech, growing daily
  • Labelled in-house by an ASR council — token-level consensus
  • No vendor corpora, no per-minute transcription

ARCHITECTURE · HARP — one model, every decoding mode

  • One checkpoint: AR anchoring + block-diffusion + self-speculation
  • Length-triggered sparse SSM-MoE → long turns stay linear-cost
  • Quantization-aware → runs on a single GPU
~7× streams per GPU vs AR baseline

TRAINING · Train big, ship small

  • Teachers we never serve (Indic LLM + expressive TTS) generate unlimited data
  • Distill big→small, then heavy human-in-the-loop RL across 31 languages
Small model ≈ capability of one 2–3× its size

COST · Compute is the only bill

  • Data, ASR, generation, RL judging — all in-house
  • Data prep ~85% cheaper than vendors
  • Compute is the only cost we share with a conventional lab
3–4M hrs owned audiominimal third-party data~7× streams/GPUsmall ≈ 2–3× size31 languages

Train where we spend; serve where we save.
